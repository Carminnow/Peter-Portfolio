Plan for the page: 
work from the top down, start with the header, then the nav, then the main, then the footer.
for the header, I will use an h1 tag with the text "Pete | Artist" and center it.
for the nav, I will add relative links to the about, portfolio, and contact sections.
for the main, I will split it into three sections: about, portfolio, and contact.
for the about section, I will round and add a border to the image provided, then I will add in the text provided.
for the portfolio section, I will add the text provided, and format each one of the images into article div tags 
for the contact section I will add the text provided and add an ordered list with the contact information provided.
for the footer, I will center the provided text.
